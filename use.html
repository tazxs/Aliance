Как пользоваться ботом?

Команды

/start - Запускает бота

/sendall(Только для админа) - Отправляет сообщения всем

/sendpython(Только для админа) - Отправляет сообщения Python разработчикам

/sendweb(Только для админа) - Отправляет сообщения Web разработчикам

Для начала нужно перейти на самого бота

@DzBotCodex_bot

Затем нажать на команду /start

После нужно выбрать ваше направление

Python или Web

Вот и все бот полностью активен и вы можете пользоваться им!
